## 📌 Overview

**Job Tracker** is a modern web app to track and manage your job applications efficiently with real-time insights and a clean UI.

This is not just a CRUD app — it includes:
- Data persistence
- Drag & Drop system
- Chart-based analytics
- UI/UX focus

## 📊 Features

### 🔹 Dashboard Analytics
- Doughnut chart (status distribution)
- Bar chart (comparison view)
- Smooth rendering & responsive design

### 🔹 Job Management
- Add company + role + status
- Drag & drop between stages
- Real-time updates

### 🔹 Smart Controls
- 🔍 Search jobs
- 🎯 Filter by status
- 🔃 Sort (Latest / Oldest / Company)

### 🔹 Data Handling
- 💾 LocalStorage persistence
- 📁 Export CSV

### 🔹 UI/UX
- 🌙 Dark / Light mode
- ✨ Glassmorphism cards
- ⚡ Hover animations
